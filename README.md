# 🪨📄✂️ Rock-Paper-Scissors Game

A simple command-line Rock-Paper-Scissors game built using Python. Play against the computer, which makes random choices, and see who wins based on the classic game rules.

## 🚀 Features

- Play against the computer
- Random computer moves using Python's `random` module
- User-friendly menu and input
- Input validation for invalid choices
- Displays the winner of each round
- Option to play multiple rounds
- Keeps the gameplay simple and interactive

## 🛠️ Technologies Used

- Python 3
- Random Module

## 📚 Concepts Practiced

- Variables and Data Types
- Conditional Statements (`if`, `elif`, `else`)
- Loops (`while`)
- Functions
- User Input
- Random Number Generation
- Input Validation
- Basic Game Logic

## ▶️ How to Run

1. Clone the repository.
2. Open the project folder in Visual Studio Code.
3. Run the Python file:

```bash
python rock_paper_scissors.py
```
